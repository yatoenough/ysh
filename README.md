# ysh

**ysh** is a simple shell written in Go

## Overview

This was written as a learning project to understand how shell works

Built-in commands:
- cd
- echo
- exit
- history
- pwd
- type
- which

Features:
- Navigation
- Autocompletion
- History

This project uses [readline](https://github.com/chzyer/readline) library to arrow history navigation and autocompletion

## Running

- Clone this repository
- Run ```make run```
- Have fun :D
